Web Development

Samazon

Client-server application which stores and displays products. Users view products and are able to create account or login to already existing account. Application is created using frameworks including Spring, JPA, and Thymeleaf to dynamically change what is displayed depending on currently logged- in user’s information.

Ripple

Client-server application which displays a feed of messages created by users. This application incorporates frameworks including Spring, JPA, and Thymeleaf. Users are able to: Create and post messages, Read other users’ messages, Follow other users, See who else is following current user, Option to view only messages created by current user. Using currently logged in user’s information, application will alter to reflect user’s content.
